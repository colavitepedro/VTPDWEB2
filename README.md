Este repositório contém os materiais e práticas da disciplina de Desenvolvimento Web 2 do quinto semestre do curso Bacharelado em Sistema de Informação.

## Informações do Curso

• **Instituição:** Instituto Federal de São Paulo (IFSP) - Campus Votuporanga

## Conteúdo

O repositório está organizado em pastas de práticas e atividades desenvolvidas ao longo da disciplina:

  - `index.html` - Página principal
  - `postagem.html` - Página de postagem
  - `Arquivos-Praticando1-Revisao-HTML-CSS/` - Recursos adicionais
